# Đề tài: Xây dựng mô hình ứng dụng ResNet-18 cải tiến nhận biết ảnh y khoa bất thường
Giáo viên hướng dẫn: Nguyễn Mộng Hiền
- Huấn luyện mô hình ứng dụng ResNet-18 cải tiến từ công cụ PyTorch để nhận biết khác thường trong ảnh y khoa X-quang ngực nhằm nhận biết viêm phổi trên môi trường Kaggle.
- Thiết lập dự án trên môi trường Kaggle, thêm dataset và thực hiện từng bước huấn luyện mô hình sau đó cải tiến thông số cho hiệu suất tốt hơn.
- So sánh hiệu suất khi hiệu chỉnh các chỉ số giữa mô hình trước và sau khi cải tiến có áp dụng Attention Block.
